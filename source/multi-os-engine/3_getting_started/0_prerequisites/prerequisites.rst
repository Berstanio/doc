=============
Prerequisites
=============

To download the Multi-OS Engine, open the main `Multi-OS Engine landing page <https://software.intel.com/en-us/multi-os-engine>`_.

Before installing Multi-OS Engine on your host development machine, set up your development environment.

Prerequisites for an Apple macOS host
--------------------------------------

•   macOS 10.10 or higher
•   Apple Xcode* IDE 7.0 or higher
•   Xcode Command Line Tools
•   iOS* 8 SDK or higher
•   Oracle Java* Development Kit (JDK) version 8
•   Google Android Studio* IDE 1.5 or higher
•   Android SDK with API level 9 or higher
•   Enrollment in the Apple Developer Program (required for Apple iOS* device support)

Prerequisites for a Windows* host
----------------------------------

•   A computer running Windows 7* OS x64 edition or higher
•   Oracle Java* Development Kit (JDK) version 8
•   Google Android Studio 1.5 or higher
•   Android SDK with API level 9 or higher
•   Apple iTunes* for Windows
•   Enrollment in the Apple Developer Program (required for iOS device support)
