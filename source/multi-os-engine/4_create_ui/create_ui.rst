=============================================
Creating Your App's User Interface (UI)
=============================================

The Intel's Multi-OS Engine Technology Preview provides two options for defining the layout and UI for your Apple iOS applications:

•   By using the UI designer which is available within Android Studio* after you install the Multi-OS Engine plug-in.
•   By using Apple Xcode* Interface Builder and generating Java bindings in Android Studio.
